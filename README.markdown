Dotfiles FTW
============

    cd ~
    git clone git://github.com/wookimiii/dotfiles.git
    mv dotfiles .dotfiles
    ~/.dotfiles/bin/symlink-dotfiles
    cd .dotifles
    git submodule update --init --recursive

    # You're done!
